D3 Introduction

Look around first!

Before you start, look around the code for a little bit. Skim over what we imported in the head of the HTML. We have D3.js, jquery and even bootstrap. We'll focus on D3.js for this tutorial though :). We've also linked our own css and javascript files.

You can find the CSS files in the css folder and javascript in the js folder. There's also a dataset about cereal in the data folder.

Read through index.html and index.js to proceed with the tutorial. There's a quick introduction to jquery in viz-1.js.

Let's try to add a circle to the page!

There's an svg element in the HTML below. SVG elements can contain lines, circles, rectangles, polygons and more. Using SVGs gives us very fine control of every pixel in the visualization. Look at viz-2.js to continue.

Let's try to draw something from dummy data

Here we'll make some dummy data and make a corresponding circle for every data point. There's an SVG element below. Take a look at viz-3.js.

Let's bring it from real data.

We'll open cereal.csv and add a corresponding circle for each data point. Take a look at viz-4.js

Use can use d3.csv to read a csv file, d3.json to read a json file.
